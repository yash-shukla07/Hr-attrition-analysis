# Power BI Dashboard Setup

## Dashboard Files

This folder contains Power BI dashboard files for HR Attrition Analysis.

### Dashboard Components

1. **Attrition Overview**
   - Overall attrition rate
   - Total employees
   - Trend analysis over time

2. **Department Analysis**
   - Attrition by department
   - Department-wise headcount
   - Department performance metrics

3. **Job Role Analysis**
   - Attrition by job role
   - Role-wise salary analysis
   - Role-wise satisfaction scores

4. **Employee Demographics**
   - Age distribution
   - Tenure analysis
   - Gender distribution
   - Marital status breakdown

5. **Key Insights**
   - High-risk employee segments
   - Trend analysis
   - Predictive metrics

## Instructions

1. Connect to your data source (SQL database or CSV)
2. Import cleaned data
3. Create calculated columns as needed
4. Configure visual interactions
5. Publish and share the dashboard
